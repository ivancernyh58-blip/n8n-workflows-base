# n8n Workflows Base

Большая коллекция готовых n8n-воркфлоу. Каждый файл в `workflows/` — импортируемый
n8n workflow JSON. Разложены по подпапкам по первой букве названия.

- **Всего воркфлоу:** 17679
- С AI-узлами: 7470
- С webhook-триггером: 5110
- Полный список: `index.csv`

## Импорт
n8n → Workflows → Import from File → выбрать нужный `.json` из `workflows/`.

## Безопасность
Воркфлоу с потенциальными секретами (ключи/токены/пароли) исключены.
